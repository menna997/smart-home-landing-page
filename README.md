# Smart Home Landing Page

A responsive Smart Home landing page built using HTML, SCSS, and JavaScript.

## Live Demo

https://www.frontendpro.dev/frontend-coding-challenges/smart-home-landing-page-WuSNP1EbOtFiqO48JSV0

## GitHub Repository

## Features

- Responsive design for Desktop, Tablet, and Mobile screens
- Fixed navigation bar
- Mobile navigation menu
- Hero section with call-to-action buttons
- About section
- Clients section
- Testimonials section
- FAQ accordion functionality
- Download App section
- Subscription section
- Contact page
- Footer with newsletter subscription

## Technologies Used

- HTML5
- SCSS (Sass)
- JavaScript (Vanilla JS)
- Google Fonts (Poppins)
- Unicons Icons

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd smart-home-landing-page
```

3. Open `index.html` in your browser

## Responsive Breakpoints

| Device | Width |
|----------|----------|
| Desktop | > 1024px |
| Tablet | ≤ 1024px |
| Mobile | ≤ 768px |
| Small Mobile | ≤ 600px |
| Extra Small | ≤ 320px |

## JavaScript Functionality

### Mobile Navigation

- Open mobile menu
- Close mobile menu

### FAQ Accordion

- Expand/Collapse FAQ items
- Toggle plus/minus icons

### Navbar Scroll Effect

- Adds background and shadow when scrolling
